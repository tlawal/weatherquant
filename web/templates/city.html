Adaptive Prediction Methodology

Data Source: All 5-minute METAR observations from api.weather.gov for {{ city.metar_station }} today{% if adaptive_info %} ({{ adaptive_info.kalman_n_obs }} readings since midnight){% endif %}.

Kalman Filter: 2-state Kalman filter [temp, trend] updated on every 5-min observation.{% if adaptive_info %} Current smoothed temp: {{ adaptive_info.kalman_temp }}°{{ city.unit|default('F') }}, trend: {{ "%+.2f"|format(adaptive_info.kalman_trend_per_hr) }}°/hr.{% endif %}

Regression:{% if adaptive_info %} 60-minute rolling OLS on {{ adaptive_info.kalman_n_obs }} recent points, slope: {{ "%+.2f"|format(adaptive_info.regression_slope_per_hr) }}°/hr, R²: {{ adaptive_info.regression_r2 }}. Features: {{ adaptive_info.regression_features|join(', ') }}.{% else %} 60-minute rolling OLS regression with weather features.{% endif %}

Prediction Method: Kalman trend + regression extrapolation blended by confidence, projected to each upcoming {% if observation_minutes %}:{{ observation_minutes|join(', :') }}{% endif %} observation time. A diurnal decay factor reduces the extrapolated warming rate toward zero as peak temperature time approaches (Parton & Nicholls 2012), preventing unrealistic constant-rate projections.

Diurnal Curve: When ≥6 observations are available, a parametric temperature curve (sin² rising + Gaussian falling) is fitted to the day's data, constrained by the forecast high. Near-term predictions (<2 hr) blend Kalman (responsive to real-time) with the fitted curve; far-term predictions (>2 hr) trust the physics-based curve more heavily.{% if adaptive_info and adaptive_info.diurnal_fit_active is defined %} Status: {% if adaptive_info.diurnal_fit_active %}Active{% if adaptive_info.diurnal_fit_rmse is not none %} (RMSE {{ "%.1f"|format(adaptive_info.diurnal_fit_rmse) }}°){% endif %}{% else %}Inactive (fallback to decay){% endif %}.{% endif %}

Remaining-Rise Cap: An ML-predicted ceiling (GradientBoosting trained on historical METAR) limits how much temperature can still rise above the current reading. This prevents runaway extrapolation even when the decay factor is insufficient.

Weather Factors: Process noise adapts to wind speed, precipitation, cloud cover, and humidity — unstable conditions widen prediction uncertainty (Hacker & Rife 2007). Forecast uncertainty (σ) also adjusts in real-time: overcast skies tighten ~30%, high humidity/dewpoint ~15%, active precipitation ~20%, while falling pressure widens ~25%.

Predictions recalculate every 60 seconds as new 5-minute observations arrive — always using the full day's data.
